# Data Centre Virtualization

We need to be able to apply the technologies we have discussed previously. Let us consider the benefits of virtualization in the data centre. A data centre is a complex environment with

- IT equipment (servers, switches & routers, firewalls, load balancers, storage)
- Heating Ventilation and Air Conditioning (HVAC)
- Power infrastructure
- Security and other services 

We end up with infrastructure sprawl as things grow large and unwieldy. We have a complex cost centre where we are focussed on many things, most of which are not related directly to the services we are providing. 

As we move from hardware to virtual, we can describe entire infrastructures with human/machine readable files; we achieve the _software defined data centre_ (SD-DC). As you begin to work on data centre grade tools, you reach the point where a single interface allows you to work on all the virtualized components; compute, storage and network. The single interface then gives us the ability to script and automate the entire implementation.
